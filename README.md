# Sistema de Inventario y Control de Préstamos

Aplicación web para gestión de activos corporativos y control de préstamos
de equipos, construida con JavaScript, HTML y CSS usando Claude Code como
copiloto de desarrollo.

## ¿Qué problema resuelve?
Elimina el seguimiento manual de inventario en hojas de cálculo. Permite
saber en tiempo real qué activos existen, quién tiene cada uno prestado
y desde cuándo.

## Funcionalidades
- Registro y consulta de inventario de activos
- Control de préstamos: asignación, seguimiento y devolución
- Historial de movimientos por activo y por usuario
- Interfaz limpia y operable sin capacitación técnica

## Stack
- JavaScript · HTML5 · CSS3
- Desarrollado con Claude Code (AI-assisted development)

## Cómo usar
Clonar el repo, abrir `index.html` en el navegador. No requiere servidor
ni instalación adicional.
